# Portal do Conselho Tutelar

Este é um portal informativo sobre o Conselho Tutelar, seus direitos, o ECA, Sipia-CT, perguntas frequentes e canais de denúncia. O objetivo é facilitar o acesso à informação e à proteção de crianças e adolescentes.

## Funcionalidades
- Menu superior com ícones e navegação responsiva
- Páginas: Início, Sobre, Direitos, ECA, Sipia-CT, FAQ e Denúncias
- Botões para denúncia por telefone e e-mail
- Visual moderno, responsivo e acessível

## Como usar
Basta abrir o arquivo `index.html` em seu navegador. Para publicar, envie todos os arquivos para um repositório no GitHub e ative o GitHub Pages (branch `main` ou `gh-pages`).
